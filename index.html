🤖 Configurar Servidor BiRefNet

BiRefNet — Estado da Arte em IA para Remoção de Fundo

BiRefNet (Bilateral Reference Network) produz remoção de fundo de qualidade absurda — muito superior ao remove.bg — preservando fios de cabelo, bordas transparentes e detalhes finos sem vestígio verde. Requer GPU NVIDIA (mínimo 4GB VRAM).

Como instalar o servidor BiRefNet
1
Instalar Python e dependências pip install torch transformers pillow flask flask-cors
2
Executar o servidor python server_birefnet.py O arquivo server_birefnet.py está na pasta do MEGAKEY
3
Aguardar download do modelo (~1.5GB) Apenas na primeira execução. Requer GPU NVIDIA (4-8GB VRAM)
4
Clicar em "Testar" e ativar o toggle Quando o servidor estiver rodando, o status ficará verde
